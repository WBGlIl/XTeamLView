# XTeamLView
每次启动**Teamview**都生成不一样的ID,让**Teamview**不弹窗 疑似商业活动,本项目仅仅是用于个人使用                                                                                                                               

## 使用说明:

XTvClient.exe和dll放在Teamview的安装目录下面,然后每次使用的时候先启动XTvClient.exe(开了UAC的话记得管理员权限启动)就行了

## 许可:

GPL许可

## append
每个ID大概24h后就会失效弹窗 重新启动下XTvClient就好
